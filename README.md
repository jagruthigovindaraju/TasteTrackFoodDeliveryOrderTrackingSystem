TasteTrackFoodDeliveryOrderTrackingSystem is a SQL-based Food Ordering and Restaurant Management System designed to simulate a real-world food delivery platform. The project focuses on database design, data management, and business analysis using MySQL.
The database includes interconnected tables for:
Customers,
Restaurants,
Food Menu,
Drinks Menu,
Orders,
Payments,
Feedback.
The dataset contains realistic South Indian restaurant and customer data, making it suitable for SQL practice, portfolio projects, and data analysis demonstrations.
